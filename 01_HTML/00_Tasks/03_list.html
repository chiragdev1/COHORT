Web Jargons

HTTP
Stands for Hypertext Transfer Protocol. Used for communication and transfer of data between client and server.

URL
Uniform Resource Locator. Each URL refers to a specific webpage

SSL Certificate
Stands for Secure Sockets Layer certificate. It encrypts the data transferred between client browser and website server. It enables HTTPS to show that the connection is secured.
